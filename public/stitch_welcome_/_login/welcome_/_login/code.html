OAuth Setup Required

Follow these steps to enable real Google/Microsoft login

🔵 Google Setup:

  1. Go to console.cloud.google.com
  2. Create project → Enable Google+ API
  3. Credentials → OAuth Client ID → Web App
  4. Add http://localhost:5173 to authorized origins
  5. Replace Google Client ID in code

🟢 Microsoft Setup:

  1. Go to portal.azure.com
  2. Microsoft Entra ID → App registrations
  3. New registration → Web → Add redirect URI
  4. Authentication → Enable public client flows
  5. Replace Microsoft Client ID in code
