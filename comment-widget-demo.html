How to Use the Comment Widget

Look for the floating "Expert Feedback" widget in the bottom-right corner. You can:

  • Add Comments: Share your expert insights with different priorities and categories
  • View Comments: See all feedback from other experts in chronological order
  • Admin Delete: Remove inappropriate comments (admin only)
  • Toggle Widget: Click the header to collapse/expand the widget

Integration with Other Pages

To add this widget to other workflow steps, simply include the script and set the appropriate meta tags:

<meta name="comment-page-id" content="pre_sme_questions">
<meta name="comment-session-id" content="your-session-id">
<meta name="comment-admin-mode" content="true">
<script src="./expert-comment-widget.js"></script>
